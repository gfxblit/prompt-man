# prompt-man

Vibe coding our implementation of the classic Pac-Man game.

## Goal

Create a fun, interactive, and "vibe-y" Pac-Man clone in TypeScript.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### Installation

```bash
pnpm install
```

### Scripts

- `pnpm dev`: Run TypeScript compiler in watch mode.
- `pnpm build`: Build the project.
- `pnpm start`: Run the built project.
- `pnpm lint`: Run ESLint.
- `pnpm test`: Run tests using Vitest.

## Project Structure

- `src/`: Source code and test files (inline).
- `dist/`: Compiled output.
